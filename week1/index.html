    1. Where/How should we save the functions for the buttons?

    Notes

      When you want to get the user’s input from any input or text area tag, you use the special word ‘value.’
      When you want to show something to a user, you use a section (see Section 1) and the special word ‘innerHTML.’
    Input or textarea use value, and section uses innerHTML.
